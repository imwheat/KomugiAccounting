# 小麦账本

拥有常规的一些记账功能，然后可以自定义一些自动记账规则。  
主要是给我自己用的。  

## 主要功能

- 记录支出、收入，并支持编辑、删除、退款标记。
- 支持模板管理，快速套用常用记账信息。
- 支持分类分组、成员管理、主题切换和分类图标/颜色自定义。
- 支持自动支出、自动收入订阅。
- 支持通知规则生成自动记账待办。
- 支持 JSON、CSV、XLSX 数据导入导出。

## 技术栈

- Kotlin
- Jetpack Compose
- Material 3
- Gradle / Android Gradle Plugin
